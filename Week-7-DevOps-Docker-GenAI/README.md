\# Week 7 - DevOps, Docker \& GenAI Fundamentals



\## Overview



This repository contains the Week 7 hands-on exercises completed as part of the \*\*Cognizant Deep Skilling DotNet FSE 5.0\*\* program.



The focus of this week is to understand the fundamentals of \*\*DevOps\*\*, \*\*Docker\*\*, and \*\*Generative AI\*\*, along with practical hands-on exercises.



\---



\## Objectives



\- Learn the fundamentals of DevOps.

\- Understand the Docker platform and containerization.

\- Learn basic Docker commands and workflows.

\- Understand the fundamentals of Generative AI.

\- Complete the assigned hands-on exercises.

\- Document commands and execution screenshots.

\# Docker HOL-1



\## Overview



This Hands-on Lab (HOL) focuses on learning the basic concepts of Docker and performing practical tasks using Docker commands.



The objective of this lab is to gain hands-on experience with Docker containers, images, and basic container management.



\---



\## Objectives



\- Understand Docker fundamentals

\- Learn Docker commands

\- Create and manage Docker containers

\- Work with Docker images

\- Verify container execution



\---



\## Prerequisites



\- Docker Desktop installed

\- Windows/Linux operating system

\- Basic command line knowledge



\---



\## Tasks Performed



\- Installed Docker

\- Verified Docker installation

\- Executed Docker commands

\- Created Docker containers

\- Managed Docker images

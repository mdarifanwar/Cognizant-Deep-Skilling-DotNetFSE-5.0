# Exercise 1 : E-Commerce Platform Search Function

## Objective

To compare Linear Search and Binary Search algorithms for searching products in an e-commerce platform.

## Scenario

An e-commerce platform needs an efficient search mechanism for locating products quickly.

## Technologies Used

- C#
- .NET 8
- Visual Studio 2026

## Algorithms Used

- Linear Search
- Binary Search

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|----------|--------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Conclusion

Binary Search performs much faster than Linear Search for large sorted datasets because it repeatedly divides the search space into halves.

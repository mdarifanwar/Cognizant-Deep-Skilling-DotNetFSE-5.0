\# Advanced SQL - SQL Exercise 2: Index



\## Exercise Name



\*\*SQL Exercise - Index\*\*



\---



\# Objective



The objective of this exercise is to understand the concept of SQL Indexes and how they improve database query performance. This exercise demonstrates creating, using, and managing indexes on database tables to optimize data retrieval.



\---



\# Learning Outcomes



After completing this exercise, you will be able to:



\- Understand the purpose of SQL Indexes.

\- Learn how indexes improve query performance.

\- Create indexes on database tables.

\- Execute queries using indexed columns.

\- Analyze the impact of indexes on search operations.

\- Manage indexes efficiently in SQL databases.

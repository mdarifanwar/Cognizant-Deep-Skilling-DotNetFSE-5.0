\# Exercise 5 - Return Data from a Stored Procedure



\## Objective



Create a stored procedure that returns the total number of employees in a given department.



\## Database



EmployeeManagementSystem



\## Stored Procedure



sp\_GetEmployeeCountByDepartment



\## Parameter



\- DepartmentID



\## Output



The procedure returns:



\- Department ID

\- Total Employees in that department



\## SQL Concepts Used



\- Stored Procedures

\- COUNT()

\- GROUP BY

\- WHERE Clause

\- Parameters



\## Result



The stored procedure successfully returns the total employee count for the specified department.
